Download Zip File

The button links you to a google drive folder that contains the free pdf files to download and print board game pieces. The folder contents are for personal use only. It is recommended to print on tabloid-sized cardstock for durability.

How to Get the Game

The board is available for free but does require some assembly required and additional pieces.
  1. Download pieces
  2. Print board game pieces and cards
  3. Cut out all the pieces
  4. Get 15 Character Tokens or Meeple
  5. Get 40+ Bingo Markers
  6. Get 2 dice (0-0-1-1-2-2)
  7. Gather your friends and play!
